React components are the building blocks of your UI. Understanding different ways to write and structure them makes your codebase easier to maintain and scale.

### 1. Functional Components

The most common type today, especially with hooks.

```jsx
function Welcome({ name }) {
  return <h1>Hello, {name}!</h1>;
}
```
