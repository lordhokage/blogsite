## ğŸ“ What is the `fs` Module?

The `fs` module stands for **File System**, and it allows you to **interact with the file system on your machine**. Think of it like your app's personal file butler.

There are two main ways to use it:

- **Asynchronous (non-blocking)**: Better for performance.
- **Synchronous (blocking)**: Easier to write, not so great for performance.

You import it like this:

```js
{
  `const fs = require('fs');`;
}
```
